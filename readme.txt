Sidebar widget to display recent post from specified category.

== Description ==
Sidebar widget to display recent post from specified category. This plugin also allows you to manually relate post with pages.


== Installation ==
1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ SubPanel in WordPress
3. Place `recent post flexible` and `related post flexible` in your sidebar
